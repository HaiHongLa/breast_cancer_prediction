First, we compare the total number of malignant tumour cells against the total number of benign ones from the dataset
This coefficients barplot shows how important a feature is to the machine learning model
This pairplot examines the correlation between the radius mean, texture mean, perimeter mean, and area mean between benign and malignant tumor cells. We can see the clear distinction of these cell characteristics between a cancerous tumour and a benign one.
This box plot compares the concavity mean of the cells between a malignant tumor against a benign one. The line in the middle of the box represents the median concavity mean. The upper and lower ends of the line are the maximum and minimum value of the concavity mean. We can clearly see that malignant breast tumors will typically have larger concavity mean than benign ones
Our machine learning model uses a 8 to 2 train test split, which yields this confusion matrix. The values on yellow and green box represents the odds of true positive and false negative, respectively. These values indicates that the model correctly classify the tumor cells. On the other hand, the value on the purple boxes are values of false positive, and true negative, which is when the model incorrectly classify the cells
We calculated the percentile values of every feature in our data frame, which results in this scatterplot. This plot shows that malignant cancer cells will often have characteristics in the higher percentiles, which might help the machine learning model
In this heat map, each square shows the correlation between the variables on each axis ranging from -1 to 1. Values closer to zero means there is no linear trend between the two variables. The close to 1 the correlation is, the more positively correlated they are. That is when one increases, so does the other. The closer to 1 is the value, the stronger the positive correlation is. A correlation closer to -1 is similar, but instead of both increasing one variable will decrease as the other increases. The diagonals are yellow because those squares are correlating each variable to itself (a perfect correlation). For the rest, the lower the values are, the darker the squares' color will be
